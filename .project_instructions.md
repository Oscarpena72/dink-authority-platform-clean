## SEO Editorial Control System (Added May 2026)

### New Database Fields on Article Model
- `ogTitle String?` — Open Graph title override for social sharing
- `ogDescription String?` — Open Graph description override for social sharing  
- `noindex Boolean @default(false)` — Hide from search engines
- `focusKeyword String?` — Primary keyword for SEO tracking
- `secondaryKeywords String?` — JSON string array of secondary keywords

### Existing Fields Repurposed
- `metaTitle` → Now serves as "SEO Title" in the editor (independent from editorial title)
- `metaDescription` → Now serves as "SEO Description" in the editor (independent from excerpt)

### Component Architecture
- `seo-search-control.tsx` — Standalone SEO & Search Control panel component
- Replaces the old basic collapsible SEO section in `article-form-client.tsx`
- AI-powered "Generate SEO Suggestions" button via `/api/seo-suggest` endpoint (uses LLM API)

### Metadata Cascade (in buildArticleMetadata)
- OG Title: `ogTitle` → `metaTitle` → `title`
- OG Description: `ogDescription` → `metaDescription` → `excerpt` → auto-generated
- Noindex: respects `article.noindex` boolean in robots meta

### Slug Handling
- POST: Accepts custom slug, falls back to auto-generated with timestamp suffix
- PUT: Allows slug editing (with sanitization). Warning shown for published articles.
- Uniqueness enforced at DB level and with runtime check

Se sincronizó copiando los 187 artículos reales de producción (jul-2026) + 1 artículo real extra "Megan Fudge" que solo existe en preview (el usuario confirmó que es real y NO debe borrarse). Sí tiene las columnas locale/translationOf.
- SEGURIDAD: la contraseña de la base de producción (Neon) es un secreto. NUNCA imprimirla en texto plano en respuestas, logs ni archivos. El usuario pidió rotarla. La rotación la hace el usuario en el panel de Neon y luego actualiza la cadena de conexión en las variables de entorno de su Vercel; rotarla NO afecta el entorno de preview (usan bases distintas).
