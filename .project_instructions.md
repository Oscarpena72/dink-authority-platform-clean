## Leads, CRM & Contact Registration System (April 2026)

### Data Models with Source Tracking
- **NewsletterSubscriber**: email, name?, source (default: "homepage"), createdAt
- **Subscriber**: email, phone?, name?, source (default: "article"), createdAt  
- **ContactSubmission**: name, email, message, phone?, source (default: "contact-form"), status (default: "new"), createdAt

### Admin Pages
- `/admin/newsletter` — Search, source column with colored badges, CSV export
- `/admin/subscribers` — Search, name+source columns, CSV export
- `/admin/contacts` — **NEW** expandable card view, status management (new/read/replied/archived), search, CSV export
- `/admin/leads` — **NEW** unified CRM overview: stats cards, source breakdown, recent activity timeline, links to sub-modules, Export All CSV

### APIs
- `GET /api/newsletter?format=csv` — Auth-protected, CSV export
- `GET /api/contact?format=csv` — Auth-protected, CSV export  
- `GET /api/subscribers?format=csv` — Auth-protected, CSV export
- `PUT/DELETE /api/contact/[id]` — Status update and delete

### RBAC
- `/admin/contacts` and `/admin/leads` require `super_admin` or `admin` role

### Navigation
- Sidebar order: Newsletter → Subscribers → Contacts → Leads & CRM → Users → Settings
