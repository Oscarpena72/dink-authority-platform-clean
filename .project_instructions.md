## Phase 3A — Subscriber Capture (Email + Phone)
- **New model**: `Subscriber` (id, email unique, phoneNumber optional, source, subscribedAt) — separate from existing `NewsletterSubscriber`
- **API**: `POST /api/subscribers` (public, upsert by email) + `GET /api/subscribers` (admin, supports `?format=csv` export)
- **Inline subscribe block**: Inserted at ~37% of article content (both desktop and mobile) via `SubscribeForm` component with source="article"
- **Footer subscribe form**: Added as 4th column in footer grid via `SubscribeForm` component with source="footer"
- **Admin page**: `/admin/subscribers` with table, search, refresh, CSV export button
- **Admin nav**: "Subscribers" item added with Users icon between Newsletter and Settings
