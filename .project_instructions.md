# Dink Authority Magazine - Project Instructions

## Project Overview
Professional digital sports media platform specialized in pickleball. Phase 1 MVP with architecture prepared for future growth (multi-language, multi-country, PWA, Stripe, advanced roles).

## Visual Style & Brand Identity
- **Color Palette (Updated March 2026):**
  - Primary dark: `#0A1128` (deep navy)
  - Primary dark light: `#1B2444`
  - Primary dark deep: `#060D1F`
  - Brand blue: `#0052FF` (dominant action/accent color)
  - Brand blue light: `#3377FF`
  - Brand blue dark: `#003ACC`
  - Cyan accent: `#00D4FF` (highlight/neon)
  - Orange accent: `#FF6B35` (featured badges)
  - Background light: `#F0F4F8`
  - Text gray: `#6B7B8D`
- **Typography:** Oswald (headings), Inter (body)
- **Design Language:** Modern, sporty, energetic, ESPN-inspired hierarchy. Blue is the dominant brand color for links, buttons, highlights, hover states, and navigation elements.
- **Logo:** Located at `/public/images/logo.png` - white text "DINK AUTHORITY MAGAZINE" on dark background

## Technical Architecture
- Next.js 14 App Router
- Prisma ORM with PostgreSQL
- NextAuth.js for admin authentication
- AWS S3 for media storage
- Tailwind CSS for styling
- Framer Motion for animations

## Key Pages
- `/` - Homepage with 10 sections in order: Header → Ad Banner → Hero Story → Latest News → Featured Articles → Events → Results → Magazine → Newsletter → Footer
- `/articles` - Article listing with filters/pagination
- `/articles/[slug]` - Article detail
- `/about` - About page
- `/contact` - Contact form
- `/login` - Admin login
- `/admin/*` - Admin panel (dashboard, articles, events, homepage, media, newsletter, results, magazine, settings)

## Database Models (added March 2026)
- `TournamentResult` - Tournament results with event name, location, dates, category, winners, scores
- `MagazineEdition` - Digital magazine editions with title, edition number, cover image, description, PDF link

## Homepage Sections
1. **Header** - Newsletter CTA button (anchors to #newsletter)
2. **Ad Banner** - Placeholder for sponsors
3. **Hero Story** - Large featured article
4. **Latest News** - Grid of recent articles
5. **Featured Articles** - Highlighted content
6. **Events** - Upcoming pickleball events
7. **Results** - Recent tournament results (trophy icon, scores, categories)
8. **Magazine** - Latest digital editions with cover images
9. **Newsletter** - Email signup (id="newsletter", scroll-mt-32)
10. **Footer** - Includes sponsors/partners bar (JOOLA, SELKIRK, FRANKLIN, ENGAGE)
