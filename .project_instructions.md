# Dink Authority Magazine - Project Instructions

## Project Overview
Professional digital sports media platform specialized in pickleball. Phase 1 MVP with architecture prepared for future growth (multi-language, multi-country, PWA, Stripe, advanced roles).

## Visual Style & Brand Identity
- **Color Palette (Updated March 2026 - v2):**
  - Primary indigo: `#090426` (brand-purple) — matches logo background exactly, header, footer, nav, dark sections
  - Primary indigo light: `#130A3A` (brand-purple-light) — hover/card bg on dark
  - Primary indigo dark: `#050215` (brand-purple-dark) — deepest dark
  - Pickleball green: `#39FF14` (brand-neon) — accent for buttons, hover states, labels ON DARK backgrounds
  - Green dim: `#2ED811` (brand-neon-dim) — hover variant
  - Blue: `#0052FF` (brand-blue) — secondary accent, kept for specific uses
  - Orange accent: `#FF6B35` (brand-accent) — featured badges, trophy icon
  - Background: `#F4F6F8` (brand-gray) — content section backgrounds
  - Text gray: `#6B7B8D` (brand-gray-dark)
- **CONTRAST RULES:**
  - Green (#39FF14) as TEXT only on dark backgrounds (navy, dark sections)
  - Green as BACKGROUND (with navy text) on light backgrounds
  - On light backgrounds, use navy (brand-purple) for text, green only for decorative elements (bars, borders, icons)
- **Typography:** Oswald (headings), Inter (body)
- **Design Language:** Modern, sporty, energetic, ESPN-inspired hierarchy. Navy + pickleball green is the core palette.
- **Logo:** Located at `/public/images/logo.png` - 15% larger (h-[92px] lg:w-[414px]), subtle green glow drop-shadow, no border between logo and nav
- **Hero:** Taller aspect ratio (16/9 → 16/7 responsive), larger title text (up to 7xl)

## Image Display System (added March 2026)
- **Focal Point System:** Each article has `focalPointX` and `focalPointY` (Float, default 50) in the database
- **CSS-based cropping:** Uses `object-fit: cover` + `object-position: {focalPointX}% {focalPointY}%` — no server-side image processing
- **Three display ratios:**
  - **Card (4:3):** Homepage cards (Latest News, Featured Articles), article listing grids, related articles, category pages
  - **Hero (16:9):** Homepage hero story, article detail page hero banner
  - **Social (1200×630):** OG meta tags for WhatsApp, Facebook, X, LinkedIn (uses original image URL with width/height hints)
- **Admin Focal Point Picker:** Visual click-to-set crosshair on image preview, with live previews for Card/Hero/Social ratios
- **API:** Both POST and PUT article endpoints accept `focalPointX` and `focalPointY`

## Share Buttons & Social Preview (Phase 2B - March 2026)
- **ShareButtons component** built inline in `article-detail-client.tsx` with two modes:
  - **Compact mode:** Icon-only buttons near the article title (right-aligned beside author/date)
  - **Full mode:** Labeled buttons with "Share this article" heading at the end of article content
- **Platforms:** WhatsApp, Facebook, X (Twitter), LinkedIn, Copy Link
- **Copy Link:** Visual confirmation (green checkmark + "Copied!" text) for 2.5 seconds
- **Social hover colors:** Each button has brand-correct hover color (WhatsApp green, Facebook blue, X black, LinkedIn blue)
- **OG Meta Tags:** og:title, og:description, og:type (article), og:url (full canonical URL), og:siteName, og:image (with width 1200, height 630, alt)
- **Twitter Meta Tags:** twitter:card (summary_large_image), twitter:title, twitter:description, twitter:image

## Article Sidebar (Phase 2C - March 2026)
- **Layout:** 2-column grid on desktop (`lg:grid-cols-[1fr_280px]`), single column on mobile (sidebar below article)
- **Slot 1 (Fixed):** Current magazine edition cover — auto-fetched from `MagazineEdition` where `isCurrent: true`. Clickable, links to `/magazine/[slug]` or externalUrl
- **Slot 2 & 3 (Dynamic):** Configurable sponsor/promo slots managed via Site Settings:
  - Keys: `sidebar_slot{N}_image`, `sidebar_slot{N}_link`, `sidebar_slot{N}_label`, `sidebar_slot{N}_enabled`
  - Admin config in Settings → Article Sidebar section with enable/disable toggle, image preview
- **Behavior:** Sidebar only renders if at least one slot has content. Sticky on desktop (`lg:sticky lg:top-6`)
- **Data flow:** Server-side fetch in `articles/[slug]/page.tsx` → passed as `sidebarData` prop to client component

## Database Models (added March 2026)
- `TournamentResult` - Tournament results with event name, location, dates, category, winners, scores
- `MagazineEdition` - Digital magazine editions with title, slug, edition number, cover image, description, PDF link, pdfCloudPath, pdfPageCount

## Magazine Flipbook Viewer (added March 2026)
- `/magazine/[slug]` - Built-in PDF viewer with dual mode: Flipbook (react-pageflip) + Reader
- Features: lazy loading of pages, swipe/touch support, fullscreen, zoom, keyboard navigation
- Share buttons: WhatsApp, Facebook, X, LinkedIn, Copy Link
- CTA section below viewer: Share, Latest Articles, Subscribe, Advertise
- Admin can upload PDFs via S3 presigned URLs from Magazine admin panel
- Mobile-first: defaults to Reader mode on mobile, swipe navigation, responsive
- When edition has PDF uploaded, homepage magazine section links to internal viewer
- When no PDF, falls back to external URL (heyzine)
