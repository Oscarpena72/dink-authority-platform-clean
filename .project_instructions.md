# Dink Authority Magazine

## Project Overview
Professional pickleball digital magazine platform. Phase 1 MVP with architecture ready for future growth (multiidioma, multipaís, PWA, Stripe, advanced roles).

## Tech Stack
- Next.js 14, React 18, TypeScript, Tailwind CSS, Prisma + PostgreSQL
- NextAuth.js v4 for admin authentication
- AWS S3 for media uploads
- Framer Motion for animations

## Brand Identity
- **Colors**: Purple dark (#2D1B4E), White (#FFFFFF), Neon green (#39FF14)
- **Fonts**: Oswald (headings), Inter (body)
- **Style**: Modern sports editorial, ESPN-inspired hierarchy, premium feel
- **Logo**: `/public/images/logo.png` (white text on dark purple)

## Database Schema
- Users (admin accounts)
- Articles (news/content with categories, hero/featured flags)
- Events (upcoming tournaments)
- NewsletterSubscriber (email subscribers)
- Media (uploaded files with S3 cloud storage paths)
- SiteSetting (key-value config)
- ContactSubmission (contact form entries)

## Key Pages
- **Public**: Homepage, Articles (list + detail), About, Contact, Login
- **Admin** (`/admin`): Dashboard, Articles CRUD, Events CRUD, Homepage content, Media library, Newsletter subscribers, Site settings

## Categories
news, pro-players, enthusiasts, results, events, gear, magazine, latam

## Admin Panel
- Accessible at `/admin` (requires auth)
- Manages articles, events, homepage hero/banner, media uploads, newsletter, site settings (social media URLs, WhatsApp number, etc.)
